Recover deleted photos

A 90-line bash script that wraps PhotoRec into a guided flow. No paid software. No cloud. No telemetry.

~ — recover-sd.sh

$ git clone https://github.com/vikomittum/sd-rescue.git

$ cd sd-rescue

$ ./recover-sd.sh --no-image ~/Downloads/recovered

→ Identifies the SD card

→ Confirms the device twice (safety)

→ Launches PhotoRec on the right target

→ Drops your photos into ~/Downloads/recovered

Get it on GitHub How it works
⌘

One command

No menus, no wizards. Run the script, confirm the device, watch your files come back.

∅

Zero cost

Apps like Disk Drill charge $99+ to actually recover files. This is MIT-licensed, no upsells.

⊘

Read-only

The script never writes to your SD. Both dd and PhotoRec only read. The tool asks twice before touching the device.

⌥

Two modes

Standard creates a dd image first (safer, allows retries). --no-image reads the SD directly when you're tight on disk space.

⌖

Photos only

PhotoRec recovers everything by default. Inside the TUI, filter to JPG, PNG, TIFF, and RAW formats (CR2, NEF, ARW, RAF) so you don't drown in OS junk.

⌬

Open source

Single bash file. Read it before you run it — or fork it.

Install & run

macOS only. You'll need Homebrew and a microSD reader.

1 Clone

git clone https://github.com/vikomittum/sd-rescue.git
cd sd-rescue
chmod +x recover-sd.sh

2 Plug in your SD

Insert the microSD card. macOS may show a "disk not readable" dialog — click Ignore. Don't initialize it.

3 Run

# Standard mode — creates a full image of the SD first.
# Needs free disk space equal to the SD size.
./recover-sd.sh ~/Downloads/recovered

# Or no-image mode — reads the SD directly. No extra disk space.
./recover-sd.sh --no-image ~/Downloads/recovered

4 Inside PhotoRec

The script will print the exact buttons. Quick version:

  • Partition: pick the only one shown, or [Whole disk]
  • Filesystem type: [Other]
  • Scope: [Whole] (scans deleted regions too)
  • Destination: navigate to your folder, press C

To filter only photos, open File Opt first, press s to deselect everything, then enable jpg, tif, png, and your camera's RAW format.
